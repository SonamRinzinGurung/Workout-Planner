## Fit Plan

A simple app where you can create your own workout plan and exercises to keep track of which exercises you will be hitting for the day.

Live Site - [FitPlan](https://fitplan.onrender.com/)

### Stack

- React
- TailwindCSS
- Node.js
- Express
- MongoDB
