## Fit Plan

A simple app where you can create your own workout plans. Allows drag and drop to freely customize your workout and exercise sequences as well.

Live Site - [FitPlan](https://fitplan.onrender.com/)

### Stack

- React
- TailwindCSS
- Node.js
- Express
- MongoDB
